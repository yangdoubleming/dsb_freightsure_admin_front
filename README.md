# vue-cli3.0-vueadmin
  技术栈主要使用vue-cli3.0+vue+elementUI+vuex+axios。<br>
 

 
### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and minifies for production to analyze the component percent
```
npm run analyze
```
